# Frame Extractor

This script extracts 5 frames per second from a video file using OpenCV.

## Setup

1. Install Python (3.8+ recommended)
2. Install dependencies:

```
pip install -r requirements.txt
```

## Usage

Run the script:

```
python frame_extractor.py
```

Then enter the full path to your video file when prompted.

## Output

Extracted frames will be saved in a folder called `output_frames`.

## Notes

- Works on most video formats (mp4, avi, etc.)
- If the video cannot be opened, check your file path
